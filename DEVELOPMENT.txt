# langur development
2025-12-24

I hope you find langur useful. I've put a lot into it. It started as a project for learning, not as an end in itself, but I have made drastic improvements since. The syntax is relatively settled at this point, so I'd like to get to 1.0. I only have so much time. If you'd like to contribute to the project, here are some things I'd like to see. While I don't have time for all of it, I do have ideas about how it could function and the syntax, and I'm open to input.

To make 1.0, I would like to fix setting by index. One issue is efficiency. Another is that you cannot presently set by index if using more than a single index on a variable (as shown below).

x[1] = 123      # okay
y[1][2] = 123   # doesn't work

--Anthony Davis
