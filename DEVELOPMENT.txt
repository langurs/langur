# langur development
2026-05-03

I hope you find langur useful. To this point, it's been a one-man show. It started as a project for learning and I also had many ideas about a programming language. The syntax is relatively settled. There are many optimizations that could be made.

PS. My idea for both OOP and structs is to use type definitions which can include methods, as in the possibly rough example below.


type picnicBasket {
    public val (
        nServings number
        corkscrew bool
    )

    public val happy = fn() bool {
        this.nServings > 1 and this.corkscrew
    }

    # _new method allows using type name as a constructor
    public val _new = fn(nPeople as nPeople number) picnicBasket {
        picnicBasket{nServings = nPeople, corkscrew = nPeople > 0}
    }
}

var pb = picnicBasket(nPeople = 2)

writeln pb.happy()
